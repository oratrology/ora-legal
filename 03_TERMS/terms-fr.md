## 5. CAMÉRA PASSIVE ET ANALYSE ÉMOTIONNELLE

### 5.1 Fonctionnement
Pendant les sessions de lecture actives, ORA peut activer la caméra frontale en arrière-plan pour effectuer une analyse émotionnelle en temps réel. Cette technologie détecte les états émotionnels généraux pour personnaliser votre lecture.

### 5.2 Principes Clés
| Principe | Détail |
|----------|--------|
| Usage interne uniquement | Données émotionnelles utilisées exclusivement pour personnaliser votre lecture — jamais stockées |
| Pas d'enregistrement | Aucune vidéo ou image n'est capturée, stockée ou transmise |
| Temps réel uniquement | Analyse en session seulement — aucune donnée conservée après la session |
| Optionnel | Vous pouvez refuser la permission caméra; les lectures restent disponibles |
| Pas de tiers | Données émotionnelles jamais partagées avec des tiers |
| Pas d'entraînement IA | Jamais utilisées pour entraîner des modèles d'IA |

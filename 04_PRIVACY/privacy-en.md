### 2.4 BIOMETRIC DATA — MAXIMUM PROTECTION

Hand and iris photographs = biometric data under GDPR Art.9, LOPDP Art.26, LGPD Art.11, BIPA (Illinois).

**Strict rules:**
| Rule | Detail |
|------|--------|
| Double consent | Separate explicit consent for biometrics — distinct from general account consent |
| Always optional | Text description always available as alternative — photos never mandatory |
| Purpose limitation | Used ONLY to generate your reading — never for ID verification, AI training, or profiling |
| Immediate deletion | Deleted permanently immediately after reading is generated — not retained |
| Encryption | TLS 1.3 in transit + AES-256 at rest |
| No third-party sharing | Only to Anthropic Claude API for reading generation, under DPA |
| No AI training | Explicitly prohibited |
| 24h deletion on request | Contact privacy@oratrology.com |

### 2.5 EMOTIONAL ANALYSIS — PASSIVE CAMERA DATA

During active reading sessions, ORA may use your device's front camera to perform real-time emotional analysis. This data is used exclusively to personalize your reading experience.

**How we handle this data:**
| Rule | Detail |
|------|--------|
| No recording | No video or images are captured, stored, or transmitted |
| Real-time only | Emotional state detected in-session only — not retained after session ends |
| Internal use only | Used solely to enrich your reading — never shown to you as output |
| Never shared | Emotional data never sent to third parties |
| No AI training | Never used to train AI models |
| Optional | Deny camera permission anytime via device Settings → ORA → Camera |
| No impact on access | Denying camera permission does not reduce access to any other ORA feature |

**Legal basis:**
- GDPR: Explicit consent Art. 9(2)(a) + Legitimate interest Art. 6(1)(f)
- Ecuador LOPDP: Informed consent Art. 26
- Brazil LGPD: Explicit consent Art. 11(I)

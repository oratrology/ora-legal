## 2. DADOS COLETADOS

### Dados Padrão
Identidade, dados de nascimento, fotos visuais, pagamentos (não armazenados pela ORA), dados coletados automaticamente (anonimizados).

### 2.4 DADOS BIOMÉTRICOS — PROTEÇÃO MÁXIMA
Fotos de mão e íris = dados biométricos sob LGPD Art.11, GDPR Art.9, BIPA.
Regras: duplo consentimento, sempre opcional, exclusão imediata, criptografia AES-256, sem compartilhamento, sem treinamento de IA.

### 2.5 ANÁLISE EMOCIONAL — CÂMERA PASSIVA
Durante as sessões de leitura, ORA pode usar a câmera frontal em segundo plano para análise emocional em tempo real.

| Regra | Detalhe |
|-------|---------|
| Sem gravação | Nenhum vídeo ou imagem capturado, armazenado ou transmitido |
| Somente tempo real | Detecção apenas na sessão — não retido após o término |
| Uso interno apenas | Usado para enriquecer sua leitura — nunca exibido ao usuário |
| Nunca compartilhado | Dados emocionais nunca transmitidos a terceiros |
| Sem treinamento de IA | Nunca usados para treinar modelos de IA |
| Opcional | Negar permissão de câmera não reduz acesso a outras funcionalidades |

**Base legal LGPD:** Consentimento explícito Art. 11(I) + Art. 7(IX)
